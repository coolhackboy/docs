# ElevenLabs Music

`elevenlabs-music` generates music audio from a prompt or a structured composition plan. Results are returned as task `files`, matching the standard image, video, 3D, and audio task response structure.

## Available Model

- **elevenlabs-music** - ElevenLabs Music generation with prompt and composition-plan controls

## Input Parameters

Provide exactly one creative control path:

- Use **prompt** for a single text brief.
- Use **composition_plan** for section-level arrangement control.
- Do not send both `prompt` and `composition_plan` in the same request.

All supported `input` fields:

- **prompt**: Text prompt describing the music to generate. Required when `composition_plan` is not provided. Length: `1-4100` characters.
- **composition_plan**: Structured composition plan. Required when `prompt` is not provided.
  - **positive_global_styles**: String array of global styles to include.
  - **negative_global_styles**: String array of global styles to avoid.
  - **sections**: Non-empty array of music sections.
    - **section_name**: Section name, `1-100` characters.
    - **positive_local_styles**: String array of section-specific styles to include.
    - **negative_local_styles**: String array of section-specific styles to avoid.
    - **duration**: Section duration in seconds, `3-120`. This public PoYo field uses seconds.
    - **lines**: Lyrics lines. Each line must be `200` characters or fewer.
- **duration**: Optional song length in seconds. Must be `3-600` when provided. Only valid with `prompt`.
- **is_instrumental**: When `true`, requests instrumental-only output. Only valid with `prompt`.
- **respect_sections_durations**: Controls how strictly section durations are followed. Only valid with `composition_plan`.
- **output_format**: Output format. Defaults to `mp3_44100_128`.

## Output Formats

Supported `output_format` values:

`mp3_22050_32`, `mp3_44100_32`, `mp3_44100_64`, `mp3_44100_96`, `mp3_44100_128`, `mp3_44100_192`, `pcm_8000`, `pcm_16000`, `pcm_22050`, `pcm_24000`, `pcm_44100`, `pcm_48000`, `ulaw_8000`, `alaw_8000`, `opus_48000_32`, `opus_48000_64`, `opus_48000_96`, `opus_48000_128`, `opus_48000_192`.

## Credit Billing

Billing is calculated by started minute:

- `128` credits per started minute
- Equivalent to `$0.64` per started minute
- Less than one minute is billed as one minute
- Requests with `prompt` and no `duration` are estimated as one started minute in PoYo v1

## Notes

- Provide exactly one of `prompt` or `composition_plan`.
- `duration` and `is_instrumental` can only be used with `prompt`.
- `respect_sections_durations` can only be used with `composition_plan`.
- Query status through the standard task status API, not the music detail endpoint.
- Successful responses return `files`, where the generated music uses `file_type: audio`.
