# ElevenLabs V3 TTS

`elevenlabs-v3-tts` converts text into speech audio. Results are returned as task `files`, matching the standard image, video, and 3D task response structure.

## Available Model

- **elevenlabs-v3-tts** - ElevenLabs V3 text-to-speech generation through Fal

## Required Parameters

- **text**: Text to convert to speech
  - Length: `1-5000` characters

## Optional Parameters

- **voice**: Voice name used for speech generation. Default upstream voice is `Rachel`
- **stability**: Voice stability from `0` to `1`
- **timestamps**: Boolean. When `true`, timestamps may be returned as an additional JSON file
- **language_code**: Optional language code used to enforce a language for the model
- **apply_text_normalization**: `auto`, `on`, or `off`

## Credit Billing

Billing is calculated by text length: `16` credits per `1000` characters.

## Notes

- Query status through the standard task status API, not the music detail endpoint.
- Successful responses return `files`, where the generated speech uses `file_type: audio`.
- When `timestamps` is enabled and returned by the provider, timestamps are uploaded as `timestamps.json` and returned as a separate file item with `file_type: other`.
