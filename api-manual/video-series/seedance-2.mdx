## Available Models

- **seedance-2** - Standard Seedance 2 video generation model
- **seedance-2-fast** - Faster, lower-cost Seedance 2 variant

## Key Features

- **Text-to-Video**: Generate videos directly from prompt text
- **First and Last Frame Control**: Use up to 2 images as the first and last frame
- **Multimodal Reference**: Guide generation with reference images, videos, and audio
- **Optional Audio**: Request generated audio together with the video

## Variant Notes

- `seedance-2` is the default standard-quality variant
- `seedance-2` supports `480p`, `720p`, and `1080p`
- `seedance-2-fast` uses the same input schema but is optimized for lower latency and lower credit cost, and supports `480p` and `720p` only

## Important Notes

- `image_urls` supports up to 2 items only:
  index `0` maps to the first frame and index `1` maps to the last frame.
- `image_urls` and any `reference_*_urls` fields are mutually exclusive and cannot be used together in the same request.
- `aspect_ratio` is optional. When provided, supported values are `21:9`, `16:9`, `4:3`, `1:1`, `3:4`, `9:16`.
- `duration` supports integer values from `4` to `15`.
- `resolution` supports `1080p` only for `seedance-2`. `seedance-2-fast` remains limited to `480p` and `720p`.
- `seed` is optional and can be used for reproducible generation.
- All remote URLs should be publicly accessible and directly downloadable by the upstream provider.

## Credits Reference

- Credits are calculated based on video duration and selected resolution.
- If `reference_video_urls` is provided, the final credit cost may differ from requests without video references.
