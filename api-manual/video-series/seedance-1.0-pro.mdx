# Seedance 1.0 Pro Video Generation

Generate high-quality videos using Seedance 1.0 Pro by ByteDance. Supports both text-to-video and image-to-video generation with flexible duration and resolution options.

## Available Model

- **seedance-1.0-pro** - Video generation with text-to-video and image-to-video support, plus 3x faster rendering

## Key Features

- **Flexible Modes**: Text-to-video and image-to-video
- **Multiple Resolutions**: 720p and 1080p support
- **Duration Options**: 5 or 10 seconds
- **Fast Rendering**: 3x faster than the standard Pro version
