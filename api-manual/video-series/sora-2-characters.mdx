# Sora 2 Characters

`sora-2-characters` creates a reusable character resource from a short source video. The completed task returns a character ID and the character name.

## Available Model

- **sora-2-characters** - Create a character resource from a video clip

## Required Parameters

- **video_url**: URL of the source MP4 video
- **name**: Character name, from `1` to `80` characters

## Example Request

```json
{
  "model": "sora-2-characters",
  "callback_url": "https://your-domain.com/callback",
  "input": {
    "video_url": "https://example.com/character.mp4",
    "name": "Maya"
  }
}
```

## Result Format

This model returns a character resource instead of a generated video file. The character result is returned in `files[0].file_url` as a JSON string:

```json
{
  "task_id": "task-xxx",
  "status": "finished",
  "files": [
    {
      "file_url": "{\"id\":\"char_123\",\"name\":\"Maya\"}",
      "file_type": "other"
    }
  ],
  "created_time": "2026-05-12T10:30:00",
  "progress": 100,
  "error_message": null
}
```

Parse `files[0].file_url` as JSON to read the character `id` and `name`.
