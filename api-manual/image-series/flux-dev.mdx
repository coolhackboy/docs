# Flux Dev Image Generation

Generate images with `flux-dev` from a prompt, or provide one `input.image_urls` item to edit an existing image.

Use `input.size` for text-to-image output size, `input.n` for the output image count, and `input.output_format` for the returned image format. Supported size values are `1:1`, `1:1 HD`, `4:3`, `3:4`, `16:9`, and `9:16`; `1:1 HD` is the high-definition square option. Custom `WIDTHxHEIGHT` values such as `1024x1536` are also supported. For image-to-image tasks, the source image guides the output and `input.size` is ignored.

Billing is based on output image size and output image count.

## Available Models

- **flux-dev** - Text-to-image generation and single-image editing
