# Kling-O1-Image-Edit Image Generation

`kling-o1-image-edit` is a Kling official image editing model for prompt-based image transformation.

## Available Model

- **kling-o1-image-edit** - Image editing with reference images and optional element guidance

## Supported Parameters

- `prompt` - Required editing prompt
- `image_urls` - Required reference image URLs, `1-10` images
- `elements` - Optional array of element descriptors for character/object control. Reference them in prompt as `@Element1`, `@Element2`, etc. Recommended structure: `frontal_image_url` for the main face/object image, and optional `reference_image_urls` for additional reference views
- `resolution` - `1K` or `2K`
- `size` - `auto`, `16:9`, `9:16`, `1:1`, `4:3`, `3:4`, `3:2`, `2:3`, `21:9`
- `output_format` - `jpeg`, `png`, `webp`
- `n` - Number of output images, `1-9`

## Billing Notes

Billing is based on the selected resolution and the output count specified by `n`.

## How to Call

Use the unified submit endpoint: `POST /api/generate/submit`

```bash
curl --request POST \
  --url https://api.poyo.ai/api/generate/submit \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "model": "kling-o1-image-edit",
    "callback_url": "https://your-domain.com/callback",
    "input": {
      "prompt": "Transform the product shot into a premium studio campaign image",
      "image_urls": ["https://example.com/reference1.png"],
      "elements": [
        {
          "frontal_image_url": "https://example.com/subject-front.png",
          "reference_image_urls": ["https://example.com/subject-side.png"]
        }
      ],
      "resolution": "2K",
      "size": "16:9",
      "output_format": "png",
      "n": 2
    }
  }'
```
