# Grok Imagine Image Quality Generation

Generate high-quality images using `grok-imagine-image-quality`. The same model supports text-to-image generation and image editing.

When `input.image_urls` is empty or omitted, PoYo creates images from the prompt. When `input.image_urls` has one or more images, PoYo uses those images as references for editing. Use `input.n` to set the number of output images.

Billing is image-based. The final cost depends on the requested output image count, resolution, and the number of input images.

## Available Models

- **grok-imagine-image-quality** - High-quality text-to-image and image editing
