# GPT Image 2 Generation

Generate images from text prompts or edit a single reference image with natural-language instructions using GPT Image 2 models.

## Available Models

- **gpt-image-2** - Text-to-image generation, and optional reference-image-guided generation
- **gpt-image-2-edit** - Single-image editing based on one reference image and a text instruction

## Notes

- `gpt-image-2-edit` requires `image_urls`.
- If multiple `image_urls` are passed, only the first image is used.
- Supported `size` values are only `1:1`, `2:3`, and `3:2`.
