# Flux Schnell Image Generation

Generate images with `flux-schnell` from a prompt. Use `input.size` for output size, `input.n` for output image count, and `input.output_format` for the returned image format. Supported size values are `1:1`, `1:1 HD`, `4:3`, `3:4`, `16:9`, and `9:16`; `1:1 HD` is the high-definition square option. Custom `WIDTHxHEIGHT` values such as `1024x1536` are also supported.

Billing is based on output image size and output image count.

## Available Models

- **flux-schnell** - Fast text-to-image generation
