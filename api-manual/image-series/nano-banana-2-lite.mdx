# Nano Banana 2 Lite Image Generation

Nano Banana 2 Lite is Google's fastest and most cost-efficient Gemini Image model, corresponding to Gemini 3.1 Flash Lite Image. It is designed for rapid ideation, low-latency creative drafts, high-throughput generation, and scaled image pipelines where speed and cost matter most.

## Available Models

- **nano-banana-2-lite** - Text-to-image generation
- **nano-banana-2-lite-edit** - Image editing with reference images

## Input Parameters

- `prompt` - Required. Text instruction for the generated or edited image.
- `image_urls` - Required for `nano-banana-2-lite-edit`. Not supported for `nano-banana-2-lite`.
- `size` - Optional aspect ratio. If omitted, no explicit output aspect ratio is specified.

Supported `size` values: `1:1`, `2:3`, `3:2`, `3:4`, `4:3`, `4:5`, `5:4`, `9:16`, `16:9`, `21:9`.

## Text-to-Image Example

```json
{
  "model": "nano-banana-2-lite",
  "callback_url": "https://your-domain.com/callback",
  "input": {
    "prompt": "A small red cube on a plain white background",
    "size": "1:1"
  }
}
```

## Image Editing Example

```json
{
  "model": "nano-banana-2-lite-edit",
  "callback_url": "https://your-domain.com/callback",
  "input": {
    "prompt": "Change the background to a clean pale blue studio backdrop",
    "image_urls": ["https://example.com/source-image.png"],
    "size": "1:1"
  }
}
```
